🚀 How to Run
1. Clone Repository
git clone https://github.com/SahilDhiman8072/microservices-nodejs-docker-gateway-api.git
cd microservices-nodejs-docker-gateway-api


2. Build & Run Containers
docker-compose up --build


3. Run in Background
docker-compose up -d
